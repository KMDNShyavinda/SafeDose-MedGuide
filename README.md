# SafeDose-MedGuide

SafeDose MedGuide is a web-based medicine information system built with JavaScript. It provides drug dosage guidance, side effect summaries, safety guidelines, and an AI chatbot for user support.

## Project structure

- `index.html` — main application shell
- `styles.css` — UI styling
- `app.js` — drug data, search, and chatbot logic

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app locally:
   ```bash
   npm start
   ```

Then open the local URL printed by `lite-server`.

## Features

- Search medicine names
- View dosage, side effects, and safety guidelines
- Built-in AI chatbot for support and guidance
